## Features
* Revamped personal website using NextJS' newest App router. 
* Contents are relocated to a MongoDB database and accessed through a serverless API. 

## Technologies
* NextJS
* TypeScript
* Tailwind CSS
* MongoDB
* Framer Motion
* Zustand State Management
* Jest
